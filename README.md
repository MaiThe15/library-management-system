## 1. Yêu cầu cài đặt
- Node.js (v18 trở lên)
- Git

## 2. Hướng dẫn chạy dự án 
**Bước 1:** Clone code về máy
`git clone https://github.com/MaiThe15/library-management-system.git`

**Bước 2:** Chạy Backend (Node.js)
`cd backend`
`npm install`
(Tạo file .env theo mẫu .env.example và điền DB Connection String)
`npm run dev` (Server sẽ chạy ở http://localhost:5000)

**Bước 3:** Chạy Frontend (ReactJS)
Mở một terminal mới:
`cd frontend`
`npm install`
`npm run dev` (Web sẽ chạy ở http://localhost:5173)
